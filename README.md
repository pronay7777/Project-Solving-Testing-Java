# Java Programming Exercises

This repository contains basic Java programs developed to practice **arrays, searching, lambda expressions, functional interfaces, and method references**.

## Programs Included

* Access an array element using an index
* Search an element using Binary Search
* Find the maximum element in an array
* Find the K-th smallest element
* Print all possible pairs of array elements
* Calculate the sum of even or odd digits using Lambda Expressions
* Find the N-th Fibonacci number using a Lambda Expression
* Check a Palindrome Number using Functional Interface and Method Reference
* Find the sum of the last digits of two numbers using Functional Interface and Method Reference

## Technologies Used

* Java
* Java 8 Features
* Scanner for user input

These programs demonstrate fundamental Java programming concepts through simple, user-input-based examples.
